해당 프로그램은, 2022년 소프트웨어 공학 중간고사 대체과제 프로젝트로 만들어졌습니다.

주의 사항 : map파일을 추가하고 싶으시다면 무조건 Resource 폴더 내에 map 파일을 넣으셔야 합니다.
	그렇지 않을 경우 에러가 던져집니다.

플레이 방법은 다음과 같습니다.

처음 창을 키게 되면, 플레이어의 수를 적으라고 할 때, 2명에서 4명의 값을 입력하면 게임 씬으로 넘어가게 됩니다.

게임 씬으로 넘어갔을 시, Player 1먼저 시작하게 됩니다.

각자의 턴마다 행동에는 총 2가지가 존재합니다.
1. 주사위를 굴린다
2. 쉰다

주사위를 굴릴 시 순서를 알려드리겠습니다.

1. 주사위를 굴린 값을 받는다. 
1.1 만약 이 때 Bridge Card에 의해서 움직이지 못할 경우 따로 안내를 해드릴 것입니다.
2. 굴린 만큼 텍스트 박스에 맵에 대한 경로를 입력하시면 됩니다. 이 때 무조건 전달된 숫자만큼 입력하셔야 합니다.
3. 만약 끝 지점에 1번이라도 도달한다고 하면, 플레이어의 턴은 무조건 종료됩니다.
4. 끝지점에 도달하게 된다면, 해당 플레이어의 턴은 진행 할 수 없습니다.
5. 한 명의 플레이어라도 끝 지점에 도달하게 될 시엔, 나머지 플레이어는 이전 셀로 돌아갈 수 없습니다.
(예를 들어서, Cell number가 10이라면 그 전의 Cell 에는 도달 할 수 없다는 의미입니다.
6. 다리를 건너게 된다면, 다리 카드를 얻게 되고, 해당 카드는 움직임을 제한하는데 사용될 것입니다.

쉰다고 결정 할 시 순서를 알려드리겠습니다.
1. 쉬는 것은 언제든지 가능합니다.
2. 만약 Player 가 다리카드를 가지고 있다면, 쉬는 것은 다리카드의 개수를 1개 낮추게 될 것입니다.
3. 이후 턴을 넘겨주는 역할을 맡게 됩니다.

재시작을 원하실 경우 다음과 같은 작업을 수행하십시오.

1. 왼쪽 상단에 보시면 Resource폴더 내에 존재하는 map파일들을 보실 수 있으실 겁니다.
2. 만약 해당 맵으로 재시작을 하시고 싶으시다면 해당 맵 파일을 누르시고, Restart를 눌러주십시오.

주의 : 프로그램을 동작하는 중에 맵을 삭제하시거나, 맵을 추가하시는 경우 에러가 나올 수 있습니다.
